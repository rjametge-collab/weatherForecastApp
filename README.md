# Weather Mobile App

React Native + Expo weather application using OpenWeatherMap API.

## Features

- Search weather by city
- Current weather
- Celsius/Fahrenheit toggle
- Five-day forecast
- Error handling
- Mobile friendly UI

## Setup

1. Install dependencies:

npm install

2. Add your OpenWeatherMap key in:

src/services/weatherApi.js

3. Start:

npx expo start

## Technologies

- React Native
- Expo
- Axios
- OpenWeatherMap API
